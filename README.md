# SpecArchitect Agent Skill

This repository contains the master prompt and architectural blueprint for the **SpecArchitect Agent**. 

## Overview
SpecArchitect is an elite AI specification agent designed to elicit, structure, and link comprehensive Product (PRD), Technical (TRD), and UI/UX specifications.

## Usage
Add the `SKILL.md` content to your AI agent's system prompt or skill directory to enable professional-grade specification capabilities.

## Features
- **Multi-Agent Orchestration**: Simulated sub-personas for Interviewing, Strategy, Architecture, and Design.
- **FMEA Analysis**: Built-in Risk Priority Number calculations for technical resilience.
- **Edge-Case Focus**: Exhaustive matrices for product, technical, and UI/UX edge cases.
- **Machine-Readable Blueprints**: Strictly formatted templates for downstream AI coding agents.
